# ComfyUI-ArchTone-Extractor

An open-source custom node for ComfyUI designed specifically for architectural visualization and aesthetic analysis. 

## The Problem
When generating architectural concepts, AI models often produce lighting that is too "cold" or color tones that feel "outdated" and lack a high-end, modern aesthetic. 

## The Solution
This node analyzes high-end reference images to extract key atmosphere descriptors and precise color palettes, ensuring the resulting generation matches modern architectural standards (e.g., warm cinematic lighting, specific material tones) without losing the original scale and proportion.

## Current Status
We are currently building the foundational structure. The next major update will integrate advanced local vision models to automate the extraction process.

## Installation
Clone this repository into your `ComfyUI/custom_nodes` directory:
\`git clone https://github.com/fangzhengyu0704/ComfyUI-ArchTone-Extractor.git\`
